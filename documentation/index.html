ng-keydown ng-keypress ng-keypress

keyboard events are triggerd when keys are pressed up or key down

  • ng-keydown

    is triggerd when key is pressed or key is in down position

  • ng-keypress

    is triggerd when key is pressed, it does not recognise alt, ctrl nor shift

  • ng-keyup

    is triggerd when a key is released

ng-mousedown ng-mousemove ng-mouseover ng-mouseenter ng-mouseleave ng-mouse

Mouse events are triggerd when mouse moved or buton are clicked

  • ng-mouseup

    is triggerd when mouse key is released

  • ng-mousedown

    is triggerd when key is pressed or left button is in down position

  • ng-mouseenter

    is triggerd when mouse enters in a object area that is listening the event

  • ng-mouseleave

    is triggerd when mouse leave the area object that is listening event

  • ng-mousemove

    is triggerd everytime when mouse moved from the area object that is listening event

  • ng-mouseover

    is triggerd everytime when mouse is over to the area object that is listening event
